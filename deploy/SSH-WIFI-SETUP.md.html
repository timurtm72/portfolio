🔥 WiFi ВСЕГДА В ПРИОРИТЕТЕ!

Даже когда подключен Ethernet кабель

⚠️ ВАЖНО! Этот скрипт настраивает WiFi с МАКСИМАЛЬНЫМ приоритетом!
Интернет будет работать через WiFi даже если подключен Ethernet кабель.
Ethernet будет использоваться только если WiFi недоступен.

🎯 Что делает этот скрипт:

✅ Настраивает 2 WiFi сети с автоподключением
✅ Устанавливает WiFi приоритет = 100 (максимальный)
✅ Понижает Ethernet приоритет = -10 (минимальный)
✅ Автоматически переключается между WiFi сетями
✅ Ethernet используется только если WiFi недоступен
📝 Всего 4 команды!

🚀 Пошаговая инструкция

1Перейдите в домашнюю папку

cd ~

2Создайте скрипт

⚠️ Скопируйте ВСЮ команду от cat до EOF целиком!

cat > wifi-priority-setup.sh << 'EOF' #!/bin/bash echo "=========================================" echo " WiFi ПРИОРИТЕТ над Ethernet" echo "=========================================" echo "" if [ "$EUID" -ne 0 ]; then echo "❌ Ошибка: Запустите с sudo" exit 1 fi if ! command -v nmcli &> /dev/null; then echo "📦 Установка NetworkManager..." apt-get update apt-get install -y network-manager systemctl enable NetworkManager systemctl start NetworkManager sleep 3 fi if ! nmcli device status | grep -q "wifi"; then echo "❌ Ошибка: WiFi адаптер не найден!" exit 1 fi echo "=== ПЕРВАЯ WiFi СЕТЬ ===" while true; do read -p "SSID первой сети: " SSID1 if [ -n "$SSID1" ]; then break fi echo "❌ SSID не может быть пустым!" done while true; do read -sp "Пароль первой сети: " PASS1 echo "" if [ ${#PASS1} -ge 8 ]; then break fi echo "❌ Пароль минимум 8 символов!" done echo "" echo "=== ВТОРАЯ WiFi СЕТЬ ===" while true; do read -p "SSID второй сети: " SSID2 if [ -n "$SSID2" ]; then break fi echo "❌ SSID не может быть пустым!" done while true; do read -sp "Пароль второй сети: " PASS2 echo "" if [ ${#PASS2} -ge 8 ]; then break fi echo "❌ Пароль минимум 8 символов!" done echo "" echo "Какая WiFi сеть важнее если обе доступны?" echo "1) $SSID1" echo "2) $SSID2" while true; do read -p "Выберите (1 или 2): " CHOICE if [ "$CHOICE" == "1" ] || [ "$CHOICE" == "2" ]; then break fi echo "❌ Введите 1 или 2" done if [ "$CHOICE" == "1" ]; then PRIORITY1=100 PRIORITY2=90 echo "✅ $SSID1 - приоритет 100 (главная)" else PRIORITY1=90 PRIORITY2=100 echo "✅ $SSID2 - приоритет 100 (главная)" fi echo "" echo "🔄 Удаление старых настроек..." nmcli connection delete "WiFi-1" 2>/dev/null || true nmcli connection delete "WiFi-2" 2>/dev/null || true echo "" echo "📡 Создание WiFi-1: $SSID1 (приоритет $PRIORITY1)" if nmcli connection add type wifi con-name "WiFi-1" ifname wlan0 ssid "$SSID1" wifi-sec.key-mgmt wpa-psk wifi-sec.psk "$PASS1" connection.autoconnect yes connection.autoconnect-priority $PRIORITY1; then echo "✅ WiFi-1 создан" else echo "❌ Ошибка создания WiFi-1" exit 1 fi echo "" echo "📡 Создание WiFi-2: $SSID2 (приоритет $PRIORITY2)" if nmcli connection add type wifi con-name "WiFi-2" ifname wlan0 ssid "$SSID2" wifi-sec.key-mgmt wpa-psk wifi-sec.psk "$PASS2" connection.autoconnect yes connection.autoconnect-priority $PRIORITY2; then echo "✅ WiFi-2 создан" else echo "❌ Ошибка создания WiFi-2" exit 1 fi echo "" echo "🔧 ПОНИЖАЕМ приоритет Ethernet..." WIRED_CONN=$(nmcli -t -f NAME,TYPE connection show | grep ethernet | head -1 | cut -d: -f1) if [ -n "$WIRED_CONN" ]; then nmcli connection modify "$WIRED_CONN" connection.autoconnect-priority -10 echo "✅ Ethernet '$WIRED_CONN' приоритет = -10 (минимальный)" else echo "ℹ️ Ethernet подключение не найдено" fi echo "" echo "🔄 Перезапуск NetworkManager..." systemctl restart NetworkManager sleep 5 echo "" echo "🔍 Проверка подключения..." if nmcli device status | grep -q "wlan0.*connected"; then echo "✅ WiFi ПОДКЛЮЧЕН!" echo "" echo "Активное WiFi подключение:" nmcli connection show --active | grep wifi echo "" echo "IP адрес WiFi:" ip addr show wlan0 | grep "inet " | awk '{print $2}' echo "" echo "Маршрут по умолчанию:" ip route | grep default else echo "⚠️ WiFi не подключен" echo "Доступные сети:" nmcli device wifi list fi echo "" echo "=========================================" echo " ✅ ГОТОВО!" echo "=========================================" echo "" echo "📊 Приоритеты:" echo " WiFi-1: $PRIORITY1" echo " WiFi-2: $([ $PRIORITY1 -eq 100 ] && echo 90 || echo 100)" echo " Ethernet: -10" echo "" echo "🌐 WiFi теперь ВСЕГДА в приоритете!" echo " Даже когда подключен Ethernet кабель" echo "" EOF

📌 Как скопировать:

ТРОЙНОЙ КЛИК по коду → Ctrl+C → Правая кнопка в PuTTY → Enter

3Сделайте скрипт исполняемым

chmod +x wifi-priority-setup.sh

4Запустите скрипт

sudo ./wifi-priority-setup.sh
🎉 Скрипт запустится и спросит:

1️⃣ SSID первой WiFi сети
2️⃣ Пароль первой сети (мин. 8 символов)
3️⃣ SSID второй WiFi сети
4️⃣ Пароль второй сети (мин. 8 символов)
5️⃣ Какая сеть главнее (1 или 2)

После этого WiFi будет ВСЕГДА в приоритете! ✅

🧪 Проверка приоритетов

Посмотреть все подключения и их приоритеты

nmcli -f NAME,TYPE,AUTOCONNECT,AUTOCONNECT-PRIORITY connection show

Должно быть примерно так:

  • WiFi-1 - приоритет 100 или 90
  • WiFi-2 - приоритет 90 или 100
  • Wired connection 1 - приоритет -10

Проверить маршрут по умолчанию

ip route | grep default

Должно показать: default via 192.168.X.X dev wlan0 (не eth0!)

Проверить интернет через WiFi

ping -c 4 google.com

Должно работать даже с подключенным Ethernet кабелем!

💡 Полезные команды

Если нужно вернуть приоритет Ethernet

sudo nmcli connection modify "Wired connection 1" connection.autoconnect-priority 0 sudo systemctl restart NetworkManager

Переключить WiFi сети вручную

sudo nmcli connection up WiFi-1

или

sudo nmcli connection up WiFi-2

Посмотреть все доступные WiFi

nmcli device wifi list

🔧 Решение проблем

❌ Маршрут всё равно через eth0

Выполните вручную:

sudo nmcli connection modify "Wired connection 1" connection.autoconnect-priority -10 sudo nmcli connection down "Wired connection 1" sudo nmcli connection up WiFi-1 sudo systemctl restart NetworkManager

❌ WiFi не подключается

sudo nmcli radio wifi on nmcli device wifi list sudo systemctl restart NetworkManager
