✅ Что было сделано

1. Подключение к Orange Pi

Подключились по SSH с локального компьютера (192.168.1.101)

ssh timur@192.168.1.100

2. Проект уже склонирован

Репозиторий GitHub уже находится в папке:

/home/timur/portfolio

Структура проекта:

  • backend/ - Spring Boot приложение
  • frontend/ - React приложение
  • database/ - SQL скрипты
  • docker-compose.yml - конфигурация Docker

3. Создание файла backend/.env

Создали файл с настройками базы данных:

cat > backend/.env << 'EOF'
DB_HOST=db
DB_PORT=5432
DB_USER=portfolio_user
DB_PASSWORD=portfolio_password
CORS_ALLOWED_ORIGINS=*
EOF
⚠️ Важно: Пароль БД: portfolio_password

4. Создание файла frontend/.env

Создали файл с настройками API:

cat > frontend/.env << 'EOF'
REACT_APP_API_URL=/api
EOF

5. Исправление конфигурации Docker

Проблема: Docker пытался использовать недоступное китайское зеркало

docker.mirrors.ustc.edu.cn - no such host

Решение: Удалили неправильное зеркало из конфигурации Docker:

# Очистили файл
sudo nano /etc/docker/daemon.json

# Перезапустили Docker
sudo systemctl restart docker
✅ Результат: Docker работает корректно, статус active (running)

6. Запуск сборки Docker образов

Запущена команда сборки (в процессе):

docker compose build
⏱️ Время сборки: 30-60 минут на Orange Pi Zero 3

Что происходит:

  • Скачивание образа PostgreSQL
  • Скачивание образа Java (eclipse-temurin:17-jdk-alpine)
  • Скачивание образа Node.js (node:18-alpine)
  • Скачивание образа Nginx (nginx:alpine)
  • Компиляция backend (Maven + Spring Boot)
  • Сборка frontend (npm + React)

📋 Следующие шаги (после завершения сборки)

7. Запуск контейнеров

docker compose up -d

8. Проверка статуса

docker compose ps

Должны быть запущены:

  • portfolio-db (PostgreSQL)
  • portfolio-backend (Spring Boot)
  • portfolio-frontend (React + Nginx)

9. Проверка логов

# Все сервисы
docker compose logs -f

# Только backend
docker compose logs -f backend

# Только frontend
docker compose logs -f frontend

10. Проверка работы приложения

# Проверить backend API
curl http://localhost:8080/api/projects

# Проверить frontend
curl http://localhost:3000
